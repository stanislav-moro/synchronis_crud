# Инструкция по использованию (USAGE)

## Подготовка окружения
1. Убедитесь, что установлен Python 3.10+.
2. Создайте и активируйте виртуальное окружение:
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # Linux/macOS:
   source venv/bin/activate
3. Установите зависимости:
   pip install -r requirements.txt
4. Настройте подключение к БД в файле .env или укажите свои параметры в core/config.py.


## Тестирование
Запуск всех юнит-тестов (проверка паттернов Singleton, Strategy и HTTP-маршрутов):
pytest tests/ -v
Ожидается: 6 passed, 0 failed


## Запуск сервера

### Локально (только ваш компьютер)
Запуск сервера: uvicorn main:app --reload (через терминал)
Приложение будет доступно по адресу: http://127.0.0.1:8000

### Для доступа с других устройств в локальной сети (ноутбук, телефон)
# Для запуска с ноута - запустить сервер с компа со следующими параметрами: uvicorn main:app --host 0.0.0.0 --port 8000
1. Узнайте локальный IP компьютера: ipconfig (Windows) или ip a (Linux/macOS)
2. Откройте в браузере другого устройства: http://<ВАШ_IP>:8000


## Просмотр логов
- Логи пишутся в реальном времени в файл: logs/synchronis_log.txt
- В браузере доступна временная страница для просмотра: http://127.0.0.1:8000/logs
- Паттерн Singleton гарантирует единую запись без конфликтов и дублирования экземпляров.
